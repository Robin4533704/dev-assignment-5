Question-2: What is the difference between innerHTML, innerText, and textContent ?


innerHTML: Shows or sets the entire content including HTML code. That is, it shows everything with tags

innerText:It shows or sets only the text that is visible on the screen. Text hidden with CSS is not captured here. Text is rendered according to line breaks or styles.

textcontent:It shows or sets all the text of the element, Want it to be hidden or visible. No HTML tags, just plain text.


Question-3: What is event delegation in the DOM?

Event Delegation:Event delegation is a JavaScript technique that uses event propagation to attach a single event listener to a parent element


Question-4: What is event bubbling in the DOM?

Event bubbling: Event bubbling is a process of events in the DOM, where when an event occurs in a child element (such as a click), that event gradually spreads to the top of the entire DOM (up to the root) through its parent, grandparent.
